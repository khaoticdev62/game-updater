# Product Guidelines: Sims 4 Updater Rebuild

## Visual Identity & UX
- **Aesthetic:** Modern and Minimalist. The interface should focus on high contrast, clean layouts, and a modern design system to ensure data is easily digestible.
- **Accessibility:**
    - Support for standard high-contrast themes.
    - Full keyboard navigation (Tab-cycling, shortcuts) for all primary actions.

## Communication & Tone
- **Voice:** Friendly and Guiding. The application should use approachable language to help users through technical processes like updating and repairing.
- **Task Feedback:**
    - **Visual Indicators:** Smooth, animated progress bars and clear "Estimated Time Remaining."
    - **Detailed Transparency:** A live, expandable log for power users to view file-level operations.
    - **System Awareness:** Native notifications for task completion or critical events.

## Error Handling & Reliability
- **Clarity:** Every error message must be actionable, offering a clear explanation and a suggested next step (e.g., "Retry," "Check Internet").
- **Visual Severity:** Categorize messages using distinct visual styles for Information, Warnings, and Errors.
- **Troubleshooting Support:** Provide easy-to-use tools for exporting logs to assist with community-based support.
